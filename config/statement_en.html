Capacitated Vehicle Routing

In the Vehicle Routing Problem, you must serve a set of customers with a fleet of vehicles.
In the Capacitated variant, vehicles have a limited capacity.
You must find the shortest possible set of routes.

  Rules

  • The vehicles start from the depot (index 0) and must return to it
  • Every customer should be visited exactly once.
  • For each route, the sum of the demands of the customers visited must not exceed the capacity of the vehicles.
  • The distance between two points is Euclidean - dist(a, b) = sqrt((xa - xb)2 + (ya - yb)2)

  Game Input

The program must first read the given inputs, then output a single line representing the vehicle routes.
Input

Line 1: an integer N, the number of customers

Line 2: an integer K, the number of vehicles

Line 3: an integer C, the capacity of the vehicles

Next N lines: 4 integers id X Y D, where id is the index of the customer, X and Y are the coordinates, and D is the demand

Output

A single line containing the tours separated by a semicolon

Each tour must be the indices of the customers separated by a space

The depot (0) should not be included in the output

Constraints
5 <= N <= 1001
X >= 0
Y >= 0

  Example

Given the input of the first test case :
 5 4 10  -> N = 5, K = 4, C = 10
 0 0 0 0    -> depot at (0,0) - no demand
 1 0 10 3   -> customer 1 at (0,10) demand=3
 2 -10 10 3 -> customer 2 at (-10,10) demand=3
 3 0 -10 3  -> customer 3 at (0,-10) demand=3
 4 10 -10 3 -> customer 4 at (10,-10) demand=3
Some example outputs in the correct format :
  • 1 2 3;4
    The first vehicle goes 0 -> 1 -> 2 -> 3 -> 0. The second vehicle goes 0 -> 4 -> 0
    The distance is dist(0, 1) + dist(1, 2) + dist(2, 3) + dist(3, 0) + dist(0, 4) + dist(0, 4) = 10 + 10 + sqrt(500) + 10 + sqrt(200) + sqrt(200) ~= 80.6
  • 4 2 1 3
    The first vehicle visits all 4 customers in this order.
    This solution is invalid : the sum of demands is 3 + 3 + 3 + 3 > C = 10
  • 1;2 4;3 2
    This solution is invalid : Customer 2 is visited twice
  • 1;3 4
    This solution is invalid : Customer 2 is not visited
  • 1 2;3 4
    This solution is valid and optimal. The distance is dist(0, 1) + dist(1, 2) + dist(2, 0) + dist(0, 3) + dist(3, 4) + dist(4, 0) = 10 + 10 + sqrt(200) + 10 + 10 + sqrt(200) ~= 68.3

  Note

  • The VRP is a classical NP-Hard problem. Finding an optimal solution is incredibly difficult, so you should use approximate algorithms.
    The test instances are from the CVRPLib (except the 3 first ones). You are welcome to compare your test results to the known optima. Validation instances will be similar but different to prevent hard-coded solutions.
