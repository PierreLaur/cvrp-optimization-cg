read k:int
read c:int
read n:int
loop n read index:int x:int y:int demand:int
write 1 2 3;4

STATEMENT
Challenge yourself with this classic NP-Hard optimization problem !

INPUT
k: The number of vehicles
c: The capacity of the vehicles
n: The number of customers
index: The index of the customer (0 is the depot)
x: The x coordinate of the customer
y: The y coordinate of the customer
demand: The demand

OUTPUT
A single line containing the tours separated by a semicolon (;)
Each tour must be the indices of the customers separated by a space ( )
The depot (0) should not be included in the output
